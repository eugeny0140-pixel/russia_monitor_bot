# Russia Monitor Telegram Bot

Автоматически отслеживает англоязычные аналитические источники (think tanks, СМИ) и публикует в Telegram статьи по темам:
- Россия, Украина, Путин, Кремль
- Санкции, энергетика, НАТО, ЕС, США

## Функции
- Фильтрация по ключевым словам
- Перевод заголовка и лида на русский (Google Translate + fallback)
- Дедупликация через Supabase
- Совместимость с Render (Web Service)

## Требования
- Python 3.9+
- Supabase аккаунт
- Telegram-бот и канал

## Установка
1. Клонируйте репозиторий
2. Заполните `.env` (для локального теста) или переменные в Render
3. Задеплойте на Render как **Web Service**

См. `SUPABASE_SETUP.md` для настройки базы.
